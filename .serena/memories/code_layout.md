# Code Layout

## Main Files

- `src/walking/__init__.py` - Package initialization
- `src/walking/cli.py` - Command-line interface with click commands (generate, cluster, visualize)
- `src/walking/models.py` - Data models for sessions, records, and clustering results
- `src/walking/generate.py` - Sample data generation for testing
- `src/walking/bipartite_clustering.py` - Core clustering algorithm implementation
- `src/walking/visualize_clusters.py` - Visualization functions for cluster distribution and secret spots
- `pyproject.toml` - Project configuration and dependencies
- `README.md` - Documentation in Japanese
